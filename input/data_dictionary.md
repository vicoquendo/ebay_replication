# Data Dictionary: PaidSearch.csv
| Variable | Description |
|----------|-------------|
| `date` | Date of the observation (daily frequency, April–July 2012) |
| `dma` | Designated Market Area identifier (numeric code for a geographic region) |
| `treatment_period` | 0 = pre-treatment (before May 22, 2012); 1 = post-treatment (May 22 onward) |
| `search_stays_on` | 0 = treatment group (paid search turned off after May 22); 1 = control group (paid
